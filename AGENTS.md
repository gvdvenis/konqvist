## Agent skills

### Issue tracker

Issues for this repo are tracked in GitHub Issues via `gh`. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses the canonical labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses single-context domain docs (`CONTEXT.md` at root and ADRs under `docs/adr/`). See `docs/agents/domain.md`.
